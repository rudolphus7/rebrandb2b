Setup Instructions

If you see an "Invalid Scopes" error, check your App Type:

  1. Go to Facebook Developers
  2. Create New App → Select Other → Next
  3. Select Business (Manage a business...)
  4. Do NOT select "Consumer" or "None".
  5. Use the new App ID to initialize.
